mysqldump: insufficient privileges to show create function

如果你遇到此問題，是因為登入 MySql 的使用者權限不足，假設登入的是 user01，而 user01 只有在某個 Database 有所有權限是不夠的。

請替 user01 增加預儲程序的權限
